### Деплой
Frontend можно отдавать как статику из корня проекта. Backend запускается из папки `backend`:

```bash
pip install -r backend/requirements.txt
uvicorn backend.main:app --host 0.0.0.0 --port 8000
```

На сервере укажите `BOT_TOKEN`, `DATABASE_PATH` и `CORS_ORIGINS` в `.env`.
